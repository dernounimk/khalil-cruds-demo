# 📦 Khalil CRUDS – Product Management System

A simple yet powerful **CRUD (Create, Read, Update, Delete)** web application for managing products. Built with vanilla JavaScript, HTML, and CSS. This app allows users to add, display, update, search, and delete products with automatic total price calculation.

🌐 **Live Demo**: [dernounimk.github.io/khalil-cruds-demo](https://dernounimk.github.io/khalil-cruds-demo/)

---

## ✨ Key Features

- **➕ Create (Add Product)** – Add new products with:
  - Title
  - Price
  - Taxes
  - Ads cost
  - Discount
  - Category
  - Automatic total calculation (Price + Taxes + Ads - Discount)
- **📖 Read (Display Products)** – View all products in a responsive table.
- **✏️ Update** – Edit existing product details.
- **🗑️ Delete** – Remove a single product or delete all products at once.
- **🔍 Search** – Search products by title or category.
- **✅ Data Validation** – Ensures all fields are filled before adding/updating.
- **💾 Local Storage** – Products are saved in browser's local storage (data persists after page refresh).
- **📱 Responsive Design** – Works on desktop, tablet, and mobile devices.
